# MVP 개발 범위와 로드맵

## 1차 개발

목표: 수동 입력과 업로드만으로 실제 광고주 운영이 가능한 버전을 만듭니다.

필수 기능:

- 로그인
- 역할 기반 권한
- 광고주 계정 생성
- 브랜드 생성
- 광고주별 대시보드
- 월간 KPI 목표 입력
- KPI 실적 입력
- KPI 달성률 계산
- 보고서 업로드 또는 링크 등록
- 일정 캘린더와 리스트
- 액션 플랜 입력
- 내부 메모와 공개 코멘트 분리
- 광고주 화면에서 본인 데이터만 조회

1차 제외 항목:

- 외부 광고 플랫폼 자동 연동
- 키워드 순위 자동 수집
- 고급 차트 자동 분석
- 알림 자동화
- 복잡한 승인 워크플로우

## 2차 개발

목표: 채널별 분석과 키워드 인사이트를 확장합니다.

확장 기능:

- 채널별 광고 성과 상세 입력
- 캠페인별 성과 테이블
- 소재별 성과 테이블
- 키워드 등록과 순위 변화 관리
- 자동 그래프
- 인사이트 입력 템플릿
- 주간 보고서 생성 보조
- 월간 보고서 생성 보조
- 광고주 확인 요청 상태 관리

## 3차 개발

목표: 외부 데이터 연동과 자동화를 붙입니다.

확장 기능:

- 네이버 광고 데이터 연동
- 쿠팡 광고 또는 판매 데이터 연동
- 메타 광고 데이터 연동
- 구글시트 연동
- 엑셀 업로드 파싱
- 자동 KPI 업데이트
- 알림 기능
- 보고서 자동 생성 초안
- 데이터 변경 이력과 감사 로그 강화

## 개발 순서 제안

1. 프로젝트 초기화
2. 디자인 토큰과 레이아웃 구현
3. 인증과 권한 구조 구현
4. 광고주, 브랜드, 사용자 관리 구현
5. 대시보드 샘플 데이터 구현
6. KPI CRUD 구현
7. 보고서와 파일 관리 구현
8. 일정 관리 구현
9. 액션 플랜 구현
10. DB 연결과 실제 데이터 전환
11. 관리자 운영 화면 정리
12. 광고주 화면 QA

## MVP 성공 기준

- 관리자가 광고주를 생성하고 KPI, 보고서, 일정을 등록할 수 있다.
- 광고주는 로그인 후 본인 브랜드 대시보드만 확인할 수 있다.
- 대시보드에서 이번 달 성과와 다음 액션을 1분 안에 이해할 수 있다.
- 보고서 파일과 외부 링크를 날짜, 유형, 채널 기준으로 찾을 수 있다.
- 내부 메모가 광고주 화면에 노출되지 않는다.
