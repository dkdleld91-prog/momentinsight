- 판매자 상품ID와 네이버 API 상품ID를 분리하고 첫 일치 후에도 300위 범위를 완주하도록 보강했다.

- 카드 디자인은 유지했고 정확 상품 `상품 열기`는 입력한 브랜드 상품 URL로 고정했다.
- 전체 품질검사 2차, Production 운영팀·광고주 재검증과 배포를 완료했다.
- 오가닉 상품 노출 카드를 컴팩트 프리미엄 레이아웃으로 조정하고 관련 원부를 네이버 민트·그린 톤으로 변경했다.

- Production에서 관리자·광고주 실조회, 데스크톱·390px 반응형, 콘솔 오류와 두 링크 목적지를 확인하고 운영 별칭 배포를 완료했다.
- 운영팀·광고주 사이드바 정보 구조와 프리미엄 스타일 통일, Production 배포, 실제 광고주 데스크톱·390px 모바일 검증을 완료했다.

- 코드와 전체 품질검사는 통과했지만 300개 완주 실조회 전이므로 배포 완료로 처리하지 않는다.

## 배포 중단 조건

- 정확 상품ID가 아닌 카탈로그 또는 다른 판매자 상품이 대상 순위로 표시됨
- 브랜드·카테고리가 다른 원부가 대표 순위로 선택되거나 대상 상품ID가 덮어써짐
- 브랜드 또는 상위 카테고리가 다른 상품이 관련 원부 카드로 표시됨
- 광고 결과가 오가닉 순위에 포함됨
- 관리자·광고주 어느 한쪽에서 관련 원부·정확 상품 카드 구분이 누락됨
- 데스크톱·모바일 겹침, 잘림, 가로 스크롤 또는 콘솔 오류 발생
- 전체 품질검사나 실조회 실패
- 대표 순위가 원부·정확 상품 중 숫자가 더 낮은 오가닉 순위와 다름
- 40개 보기 페이지 계산이 `ceil(rank/40)` 및 페이지 내 `((rank-1)%40)+1`과 다름
- 상품명이 키워드 검색 결과가 아닌 상품 상세로 연결되거나 `상품 열기`가 상품 상세 이외의 주소로 연결됨
- 판매자 상품 조회가 결과 링크의 판매자 상품ID가 아닌 API `productId`만으로 일치함
- 300위 선택 조회가 정확 상품 발견 지점에서 중단되어 `checkedCount`가 300보다 작음
- 플레이스 외부 응답에 네이티브 PC 오가닉 source·rankEvidence가 없거나 서로 불일치함
