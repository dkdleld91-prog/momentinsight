# 제품 요구사항

## 서비스 개요

모먼트 인사이트는 마케팅 대행사가 광고주별 성과 데이터를 정리하고, 보고서와 일정, 실행 전략을 광고주에게 공유하기 위한 통합 관리 플랫폼입니다.

단순 파일 업로드 도구가 아니라 광고주가 `현재 상태`, `성과 변화`, `문제 지표`, `다음 실행 방향`을 한 화면에서 이해할 수 있는 운영형 SaaS를 목표로 합니다.

## 사용자

### 관리자

대행사 내부 사용자를 의미합니다.

주요 작업:

- 광고주 계정 생성
- 브랜드 및 권한 관리
- KPI 목표 설정
- 광고 성과 입력
- 보고서 업로드
- 일정 등록
- 내부 메모 작성
- 광고주 공개 코멘트 작성
- 광고주별 상태 관리

### 광고주

본인 브랜드의 성과와 보고서를 확인하는 외부 사용자를 의미합니다.

주요 작업:

- 메인 대시보드 확인
- KPI 진행률 확인
- 보고서 열람
- 일정 확인
- 파일 다운로드 또는 링크 접근
- 액션 플랜과 확인 요청 사항 확인

## 핵심 화면 흐름

1. 핵심 요약
