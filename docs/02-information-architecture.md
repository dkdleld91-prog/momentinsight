# 정보 구조와 메뉴

## 권한 구조

| 역할 | 범위 | 설명 |
| --- | --- | --- |
| Super Admin | 전체 | 전체 광고주, 사용자, 설정, 데이터 접근 |
| Manager | 할당 광고주 | 담당 광고주 데이터 관리 |
| Analyst | 할당 광고주 | 성과, KPI, 키워드, 보고서 입력 |
| Client Owner | 본인 브랜드 | 광고주 대표 사용자 |
| Client Viewer | 본인 브랜드 | 읽기 전용 광고주 사용자 |

## 데이터 접근 원칙

- 관리자 계정은 역할과 할당 범위에 따라 광고주 데이터를 본다.
- 광고주 계정은 연결된 `client_id`, `brand_id` 데이터만 본다.
- 내부 메모는 관리자 역할만 접근한다.
- 공개 코멘트는 광고주 화면에 노출 가능하다.
- 파일은 공개 여부와 권한 범위에 따라 접근을 제한한다.

| 메뉴 | 대상 | 목적 |
| --- | --- | --- |
| Dashboard | 관리자, 광고주 | 현재 성과와 다음 액션 요약 |
| KPI Report | 관리자, 광고주 | 목표 대비 실적과 달성률 관리 |
| Sales Report | 관리자, 광고주 | 월별, 채널별 매출 흐름 확인 |
| Ads Performance | 관리자, 광고주 | 네이버, 쿠팡, 메타 등 채널 성과 분석 |
| Keyword Analysis | 관리자, 광고주 | 핵심 키워드 순위와 검색 성과 분석 |
| Weekly Report | 관리자, 광고주 | 주간 성과와 실행 내용 공유 |
| Monthly Report | 관리자, 광고주 | 월간 성과와 다음 달 전략 공유 |
| Schedule | 관리자, 광고주 | 일정 캘린더와 리스트 |
| Files | 관리자, 광고주 | 업로드 자료와 외부 링크 관리 |
| Action Plan | 관리자, 광고주 | 다음 실행 계획과 확인 요청 사항 |
| Client Management | 관리자 | 광고주, 브랜드, 사용자, 상태 관리 |
| Admin Settings | 관리자 | 권한, 채널, 보고서 유형, 시스템 설정 |

## 광고주 화면 구조

광고주가 접속하면 `Dashboard`가 기본 화면입니다.

구성 순서:

1. 현재 상태 요약
2. 이번 달 KPI 진행률
3. 핵심 지표 카드
4. 채널별 성과 카드와 차트
5. 키워드 순위 요약
6. 이번 주 핵심 인사이트
7. 다음 액션 플랜
8. 일정과 확인 요청 사항
9. 최근 보고서와 파일

## 관리자 화면 구조

관리자는 광고주 선택이 모든 기능의 기준입니다.

구성 순서:

1. 전체 광고주 상태 요약
2. 광고주 검색과 필터
3. 광고주별 KPI 상태
4. 보고서 업로드 현황
5. 일정과 제출 마감
6. 확인 필요 코멘트
7. 내부 메모와 작업 상태
8. 광고주 상세 관리
