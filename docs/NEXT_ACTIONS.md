## 배포 중단 조건

- 정확 상품ID가 아닌 카탈로그 또는 다른 판매자 상품이 대상 순위로 표시됨
- 관련 원부가 정확 상품의 대상 ID 또는 현재 순위를 변경함
- 브랜드 또는 상위 카테고리가 다른 상품이 관련 원부 카드로 표시됨
- 광고 결과가 오가닉 순위에 포함됨
- 관리자·광고주 어느 한쪽에서 관련 원부·정확 상품 카드 구분이 누락됨
- 데스크톱·모바일 겹침, 잘림, 가로 스크롤 또는 콘솔 오류 발생
- 전체 품질검사나 실조회 실패
