# 페이지별 기능 정의

## Dashboard

목적: 광고주가 현재 브랜드 성과와 다음 실행 방향을 가장 빠르게 이해하는 화면입니다.

주요 섹션:

- 현재 상태 요약
- 핵심 KPI 카드
- 목표 대비 달성률
- 전월 대비 변화율
- 채널별 성과 요약
- 키워드 순위 요약
- 진행 중인 작업
- 이번 주 핵심 인사이트
- 다음 액션 플랜
- 최근 보고서와 일정

관리자 기능:

- 광고주 선택
- 공개 코멘트 입력
- 내부 메모 입력
- 주요 인사이트 수정
- 액션 플랜 상태 변경

## KPI Report

목적: 월간 목표와 실제 실적을 비교합니다.

주요 기능:

- 월별 KPI 목표 설정
- 실제 실적 입력
- 달성률 자동 계산
- 상승, 하락, 유지 표시
- KPI별 상세 메모
- 채널별 KPI 연결

핵심 표시:

- 목표 매출
- 실제 매출
- 광고비
- ROAS
- 구매 수량
- CPA
- CPC
- CTR
- CVR
- 리뷰 수
- 키워드 순위
- 검색 노출 현황

## Sales Report

목적: 매출 흐름을 월간, 주간, 채널 기준으로 확인합니다.

주요 기능:

- 월별 매출 테이블
- 전월 대비 증감
- 채널별 매출 기여도
- 구매 수량과 객단가
- 주요 변동 원인 메모

## Ads Performance

목적: 채널별 광고 성과를 비교합니다.

채널 예시:

- 네이버
- 쿠팡
- 메타
- 자사몰
- 검색광고
- 디스플레이 광고

주요 기능:

- 채널별 성과 카드
- 광고비, 매출, ROAS 비교
- 노출, 클릭, 전환 퍼널
- 캠페인별 테이블
- 소재별 성과 테이블
- 전월 대비 변화 표시

## Keyword Analysis

목적: 핵심 키워드의 순위 변화와 보완 방향을 관리합니다.

주요 기능:

- 키워드 등록
- 현재 순위와 이전 순위 비교
- 네이버, 쿠팡 순위 분리 표시
- 검색량과 노출수 표시
- CTR, 전환 기여도 표시
- 광고 노출 여부 표시
- SEO 작업 필요 여부 표시
- 강화 키워드와 보완 키워드 구분

## Weekly Report

목적: 주간 성과, 진행 작업, 다음 주 계획을 공유합니다.

주요 기능:

- 주차별 보고서 등록
- 핵심 변화 요약
- 성과가 좋은 채널
- 성과가 낮은 채널
- 개선 필요 지표
- 다음 주 실행 계획
- 광고주 확인 필요 사항
- PDF, 이미지, 엑셀, 링크 첨부

## Monthly Report

목적: 월간 성과와 다음 달 전략을 정리합니다.

주요 기능:

- 월별 보고서 등록
- KPI 달성 요약
- 매출과 광고 성과 요약
- 키워드 성과 요약
- 콘텐츠와 캠페인 결과
- 다음 달 전략
- 첨부 파일과 외부 링크

## Schedule

목적: 광고주와 공유할 수 있는 일정표입니다.

보기 방식:

- 캘린더
- 리스트

일정 상태:

- 예정
- 진행 중
- 완료
- 보류
- 확인 필요

## Files

목적: 보고서, 이미지, 엑셀, 외부 링크를 모아 관리합니다.

주요 기능:

- 파일 업로드
- 구글 드라이브 링크 등록
- 노션 링크 등록
- 광고주, 날짜, 채널, 보고서 유형 필터
- 공개 여부 설정
- 다운로드 또는 새 창 열기

## Action Plan

목적: 데이터 기반 다음 실행 계획을 명확히 보여줍니다.

주요 항목:

- 이번 주 핵심 변화
- 성과가 좋은 채널
- 성과가 낮은 채널
- 개선이 필요한 지표
- 광고비 조정 제안
- 키워드 보완 방향
- 콘텐츠 개선 방향
- 다음 주 실행 계획
- 광고주 확인 필요 사항

## Client Management

목적: 관리자용 광고주 관리 화면입니다.

주요 기능:

- 광고주 생성
- 브랜드 생성
- 광고주 사용자 초대
- 권한 설정
- 담당자 배정
- 광고주 상태 관리
- 내부 메모 관리
- 공개 코멘트 관리

## Admin Settings

목적: 운영에 필요한 기본값을 관리합니다.

주요 기능:

- 사용자 역할 관리
- 채널 목록 관리
- 보고서 유형 관리
- 일정 유형 관리
- KPI 템플릿 관리
- 파일 공개 정책 관리
